L'onglet Styles CSS de la boîte d'édition de tout élément Karuta vous permet d'en modifier l'apparence. Vous pouvez spécifier séparément l'apparence du libellé de l'élément et de son contenu. Le contenu d'un élément de structure est tous les éléments qu'il contient, alors que le contenu d'un élément ressource est la valeur de la donnée contenue. Des champs pré-définis vous permettent d'ajuster le poids, le style, la taille et la couleur de la police, l'alignement du texte, l'espace avant (padding-top) et la couleur du fond. Un champ de texte libre vous permet aussi de spécifier votre propre code CSS bien formé. Si plusieurs éléments d'un portfolio doivent avoir le même style CSS, vous pouvez déclarer une classe CSS dans l'élément Root du portfolio puis spécifier le nom de classe dans chaque élément visé.
