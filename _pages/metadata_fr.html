Code

Le code est utilisée dans les listes déroulantes (Get_Resource, Get_Get_Resource, Menu - Get_Multiple, Menu - Get_Get_Multiple, Menu - Import_Multiple) pour identifier le choix de l'usager. Par défaut le code et le libellé de l'élément apparaissent dans la liste déroulante et seul le libellé apparait ensuite dans la ressource.

  • Le mot-clé @ ajouté au code rendra le code non visible dans la liste d'une boîte de choix.
  • Le mot-clé # ajouté au code rendra le code visible dans la ressource.
  • Le mot-clé & ajouté au code (uniquement pour l'élément Item) rendra la valeur associée visible dans la ressource.
  • Le mot-clé % ajouté au code rendra le libellé non visible dans la ressource.
  • Le mot-clé $ ajouté au code rendra le libellé non visible dans la liste d'une boîte de choix.

Query

Query est la fonction de recherche utilisée par Get_Resource, Get_Get_Resource, Menu - Get_Multiple, Menu - Get_Get_Multiple, Menu - Import_Multiple ou encore dans les Sections, Pages ou sous-sections.

Format Get_

code_portfolio.semantictag.label

Format Get_Get_

code_portfolio.position.semantictag.label

  • code_portfolio : code du portfolio référentiel où chercher
  • position : Mots réservés. Indique où chercher dans le portfolio courant le code du parent permettant de sélectionner dans le référentiel. Prend une des valeurs suivantes: child, sibling, parent, parent.parent, parent.parent.parent
  • semantictag : tag sémantique des élements à chercher
  • label : Mot réservé

Menu - Get_Multiple

Format

function,get_multiple/query/code_portfolio.semtag/get_resource_semtag,libellé@fr,role

  • function : mot réservé
  • get_multiple : mot réservé
  • query : query du Get_Resource
  • code_portfolio.semtag : part à ajouter (peut être juste le Get-Resource)
  • get_resource_semtag : tag sémantique du Get_Resource (dans part à ajouter)
  • libellé@fr : Libellé du menu
  • role : rôle(s) voyant le menu

Query

Query est la fonction de recherche utilisée par Get_Resource, Get_Get_Resource, Menu - Get_Multiple, Menu - Get_Get_Multiple, Menu - Import_Multiple ou encore dans les Sections, Pages ou sous-sections.

Format Get_

code_portfolio.semantictag.label

Format Get_Get_

code_portfolio.position.semantictag.label

  • code_portfolio : code du portfolio référentiel où chercher
  • position : indique où chercher dans le portfolio courqnt le code du parent permettant de sélectionner dans le référentiel. Prend une des valeurs suivantes: child, sibling, parent, parent.parent, parent.parent.parent
  • semantictag : tag sémantique des élements à chercher
  • label : Mot réservé

Menu - Get_Get_Multiple

Format

function,get_get_multiple/query/code_portfolio.semtag/get_resource_semtag,libellé@fr,role

  • function : mot réservé
  • get_get_multiple : mot réservé
  • query : query du Get_Get_Resource
  • code_portfolio.semtag : part à ajouter (peut être juste le Get-Resource)
  • get_resource_semtag : tag sémantique du Get_Resource (dans part à ajouter)
  • libellé@fr : Libellé du menu
  • role : rôle(s) voyant le menu

Menu - Import_Multiple

Format

function,import_multiple/query/code_portfolio,libellé@fr,role

  • function : mot réservé
  • import_multiple : mot réservé
  • query : query du Get_Resource - retourne une liste de code
  • code_portfolio : code du portfolio duquel on importe le noeud avec le(s) code(s) sélectionné(s) - peut être ou pas le même que celui du Get_resource
  • libellé@fr : Libellé du menu
  • role : rôle(s) voyant le menu

Menu - Import_GGMultiple

Format

function,import_ggmultiple/query/code_portfolio,libellé@fr,role

  • function : mot réservé
  • import_ggmultiple : mot réservé
  • query : query du Get_Get_Resource - retourne une liste de code (le code du portfolio peut-être le mot-réservé '?', dans ce cas le code du portfolio sera le code de la sélection du parent)
  • code_portfolio : code du portfolio duquel on importe le noeud avec le(s) code(s) sélectionné(s) - peut être ou pas le même que celui du Get_resource (le code du portfolio peut-être le mot-réservé '?', dans ce cas le code du portfolio sera le code de la sélection du parent)
  • libellé@fr : Libellé du menu
  • role : rôle(s) voyant le menu

Exemple d'utilisation: un Get-Resource permet de sélectionner un référentiel parmi une liste de référentiels (le code de l'item doit être le code du portfolio du référentiel). Un Import_GetGetMultiple permet d'importer des éléments du référentiel sélectionné dans le Get_resource.

query :

function,import_ggmultiple/?.child.referentiel.skill$.label/?,Ajouter une compétence@fr,auteur

Remarque: Ce menu se trouve sur une unitstructure et le Get_resource est en-dessous d'où le child, referentiel est le tag du Get_resource ainsi que des items de la liste de référentiels. Chaque item a pour code le code du portfolio référentiel. Les points d'interrogation ? font référence à ce code de pprtfolio référentiel
